This folder contains the following files with cleaned up 
data from macrophage ATAC-seq study:

1. ATAC_consensus_peaks.gff3 - Consensus peak calls accross all conditions in the GFF3 format.
2. peak_accessibility_counts.txt.gz - Raw counts of fragments overlapping each peak in each sample.
3. peak_accessibility_cqn.txt.gz - Raw counts normalised using the cqn Bioconductor packages. 
	Corrected for peak length and peak GC content.
4. peak_metadata.txt - metadata regarding each individual peak.
5. sample_metadata.txt - metaata regarding each individual sample in the dataset.
